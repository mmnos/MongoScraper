# MongoScraper

## Overview

This app allows the user to scrape the ESPN website and displays the current headlines. User's are allowed to click the link to be taken to the article, delete the article, or add a comment. 

## Technology

* HTML/CSS
* Materialize
* Handlebars
* Node
* Express
* MongoDB
* Axios
* Cheerio
